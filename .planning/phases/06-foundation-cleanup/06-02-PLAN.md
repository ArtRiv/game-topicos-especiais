In `src/networking/network-manager.ts`, add after the `disconnect()` method (around line 108):

```typescript
/**
 * Tears down the WebRTC mesh (all peer connections and data channels) while
 * keeping the socket.io signaling connection alive. Used when returning to
 * lobby after a match — enables rematch without full reconnect (FND-04).
 */
teardownMesh(): void {
  this.stopGameTick();
  this.#closeAllPeerConnections();
  this.#lastSentSnapshot = null;
  this.#matchPlayers = [];
  // DO NOT call this.#socket.disconnect() — keep signaling alive for lobby
}
```

Key difference from `disconnect()`: does NOT call `this.#socket.disconnect()`. Clears `#matchPlayers` (which `disconnect()` does not -- but `#closeAllPeerConnections()` already clears `#socketToPlayerId`).
