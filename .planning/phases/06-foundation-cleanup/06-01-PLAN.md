**LobbyScene refactor (primary work):**

1. Remove the `public shutdown(): void` method entirely (lines 33-40).

2. In the `create()` method, add a SHUTDOWN event listener at the end:
```typescript
this.events.once(Phaser.Scenes.Events.SHUTDOWN, () => {
  EVENT_BUS.off(CUSTOM_EVENTS.NETWORK_CONNECTED, this.#onConnected, this);
  EVENT_BUS.off(CUSTOM_EVENTS.NETWORK_LOBBY_UPDATED, this.#onLobbyUpdated, this);
  EVENT_BUS.off(CUSTOM_EVENTS.NETWORK_LOBBY_STARTED, this.#onLobbyStarted, this);
  EVENT_BUS.off(CUSTOM_EVENTS.NETWORK_DISCONNECTED, this.#onDisconnected, this);
  EVENT_BUS.off(CUSTOM_EVENTS.NETWORK_LOBBY_UPDATED, this.#onWaitingRoomUpdate, this);
  this.#isHost = false;
  this.#clearView();
});
```

Note: LobbyScene registers different handlers at different times (lobby list view vs waiting room view). The SHUTDOWN cleanup must remove ALL possible handlers regardless of which view is active. The existing internal `EVENT_BUS.off()` calls inside `#onConnected`, `#showWaitingRoomView`, and `#onLobbyStarted` remain -- they handle view-transition cleanup. The SHUTDOWN hook is the safety net for scene-level cleanup.

3. Import `Phaser.Scenes.Events` if not already imported (it's available via the `* as Phaser` import).
