    **`_onConnect()` logic:**
    ```typescript
    private _onConnect(): void {
      const ip = (this.ipInput.node as HTMLInputElement).value.trim() || 'localhost';
      const port = 3000;
      const url = ip.includes(':') ? `http://${ip}` : `http://${ip}:${port}`;
      this._playerName = (this.nickInput.node as HTMLInputElement).value.trim() || 'Player';

      const nm = NetworkManager.init(url);
      nm.connect();

      // One-time listener: when connected, fetch lobby list
      EVENT_BUS.once(CUSTOM_EVENTS.NETWORK_CONNECTED, () => {
        nm.sendLobbyList();
        this._showLobbyListView();
      });
    }
    ```

    **View B — Lobby List Screen** (`_showLobbyListView()`):
    - Destroys View A DOM elements.
    - Title: "LOBBIES"
    - Scroll area (static, up to 6 lobbies): renders each lobby as a row (name, players count, mode), clickable.
    - "CREATE LOBBY" button. On click: `NetworkManager.getInstance().sendLobbyCreate(this._playerName)`.
    - On `CUSTOM_EVENTS.NETWORK_LOBBY_UPDATED` with `{ lobbies: Lobby[] }`: re-render lobby rows.
    - On click of a lobby row: `NetworkManager.getInstance().sendLobbyJoin(lobby.id, this._playerName)`.
    - On `CUSTOM_EVENTS.NETWORK_LOBBY_UPDATED` with `{ lobby: Lobby }` (joined/created response): call `_showWaitingRoomView(lobby)`.

    **View C — Waiting Room Screen** (`_showWaitingRoomView(lobby: Lobby)`):
    - Title: "WAITING ROOM"
    - Subtitle: lobby.name
    - Player list: renders each PlayerInfo with name + colored dot (tintColor).
    - "START GAME" button (visible only if local socket is the host). On click: `NetworkManager.getInstance().sendLobbyStart()`.
    - On `CUSTOM_EVENTS.NETWORK_LOBBY_UPDATED` with `{ lobby }`: re-render player list.
    - On `CUSTOM_EVENTS.NETWORK_LOBBY_STARTED` with `{ matchConfig }`:
      ```typescript
      this.scene.stop(SCENE_KEYS.LOBBY_SCENE);
      this.scene.start(SCENE_KEYS.PRELOAD_SCENE);
      ```
      (PreloadScene loads assets then starts GAME_SCENE naturally.)

    **Cleanup:** `shutdown()` override → call `EVENT_BUS.off()` for all NETWORK_* listeners registered in this scene.

    Use `isHost` flag: compare `NetworkManager.getInstance().localPlayerId` with `lobby.hostId`.
