## Tasks

### Task 03-05-01 — Import new spell modules + fix `#onRemoteSpellCast`

**Action:**

**Step 1 — Named imports in `src/scenes/game-scene.ts`.**

The new spell classes need to be imported so Phaser can use them in `instanceof` checks (Task 03-05-02) AND so their `registerSpell()` side-effects fire at module load time. Named imports handle both.

**Do NOT use side-effect-only imports with `../../` paths** — all spell imports in `game-scene.ts` use `'../game-objects/spells/...'` (single `../` from `src/scenes/`). The named imports in Task 03-05-02 cover this automatically.

**Step 2 — Add `#remoteSpellGroup` field** to `GameScene` class (near other group declarations):
```typescript
#remoteSpellGroup!: Phaser.GameObjects.Group;
```

**Step 3 — Initialize `#remoteSpellGroup` in `create()`** (after `this.#spellGroup` or similar group initializations):
```typescript
this.#remoteSpellGroup = this.add.group();
```

**Step 4 — Replace the broken `#onRemoteSpellCast` handler.** Find the existing method:
```typescript
#onRemoteSpellCast = (payload: SpellCastBroadcast): void => {
  // Re-emit locally so the existing spell system handles visual effect
  EVENT_BUS.emit(CUSTOM_EVENTS.SPELL_CAST, {
    x: payload.x,
    y: payload.y,
    element: payload.element,
    direction: payload.direction,
    spellId: payload.spellId,
  });
};
```

Replace with:
```typescript
#onRemoteSpellCast = (payload: SpellCastBroadcast): void => {
  const factory = SPELL_FACTORY_REGISTRY[payload.spellId];
  if (!factory) {
    console.warn(`[GameScene] No factory for remote spellId: ${payload.spellId}`);
    return;
  }

  const spell = factory(
    this,
    payload.x,
    payload.y,
    payload.targetX ?? payload.x + 1,
    payload.targetY ?? payload.y,
    payload.direction,
  );

  this.#remoteSpellGroup.add(spell.gameObject);

  spell.gameObject.once(Phaser.GameObjects.Events.DESTROY, () => {
    this.#remoteSpellGroup.remove(spell.gameObject, false, false);
  });
};
```

**Step 5 — Add `targetX`/`targetY` to broadcast type and `#onLocalSpellCast`.**

In `src/networking/types.ts`, add optional fields to `SpellCastBroadcast`:
```typescript
targetX?: number;
targetY?: number;
```

In `src/scenes/game-scene.ts`, find `#onLocalSpellCast`. Ensure the `sendSpellCast()` call includes `targetX` and `targetY` from the spell cast event payload. Without this, all remote spells target `x + 1` (fire rightward regardless of actual aim). The `CUSTOM_EVENTS.SPELL_CAST` event payload emitted by `SpellCastingComponent` must also carry `targetX`/`targetY` — if it doesn't, add them to the emit call in `spell-casting-component.ts`.

Example `#onLocalSpellCast` update:
```typescript
nm.sendSpellCast({
  ...existingFields,
  targetX: payload.targetX,
  targetY: payload.targetY,
});
```

Also add the import at the top of game-scene.ts:
```typescript
import { SPELL_FACTORY_REGISTRY } from '../game-objects/spells/spell-registry';
```

```typescript
this.physics.add.collider(this.#remoteSpellGroup, this.#collisionLayer, (spellObj) => {
  if (spellObj instanceof IceShard) spellObj.explode();
  if (spellObj instanceof WindBolt) spellObj.explode();
  if (spellObj instanceof FireBolt) spellObj.explode();
  if (spellObj instanceof EarthBolt) spellObj.explode();

// Remote spells vs enemies (per room, same as local spells)

  const roomId = parseInt(key, 10);
  if (!this.#objectsByRoomId[roomId]?.enemyGroup) return;
  this.physics.add.overlap(
    this.#remoteSpellGroup,
    this.#objectsByRoomId[roomId].enemyGroup,
    (spellObj, enemy) => {
      const enemyGameObject = enemy as CharacterGameObject;
      if (enemyGameObject.isDefeated) return;
      if (spellObj instanceof IceShard)     { enemyGameObject.hit(DIRECTION.DOWN, spellObj.baseDamage); spellObj.explode(); return; }
      if (spellObj instanceof WindBolt)     { enemyGameObject.hit(DIRECTION.DOWN, spellObj.baseDamage); spellObj.explode(); return; }
      if (spellObj instanceof FireBolt)     { enemyGameObject.hit(DIRECTION.DOWN, spellObj.baseDamage); spellObj.explode(); return; }
      if (spellObj instanceof EarthBolt)    { enemyGameObject.hit(DIRECTION.DOWN, spellObj.baseDamage); spellObj.explode(); return; }
      if (spellObj instanceof ThunderStrike){ spellObj.hitEnemy(enemyGameObject); return; }
      if (spellObj instanceof WaterSpike)   { spellObj.hitEnemy(enemyGameObject); return; }
    },
  );
});
```

### Task 03-05-03 — PVP-01 unit test

**Action:**
Create `src/game-objects/spells/pvp-01.test.ts` to prove every element has a castable primary spell — static TypeScript + runtime proof.

```typescript
import { describe, it, expect } from 'vitest';
import { ELEMENT } from '../../common/common';
import { SPELL_SLOT_REGISTRY } from './spell-registry';

describe('PVP-01 — all elements have a primary spell (slot 0)', () => {
  it('every ELEMENT value maps to a non-null slot 0 in SPELL_SLOT_REGISTRY', () => {
    for (const element of Object.values(ELEMENT)) {
      const primarySpell = SPELL_SLOT_REGISTRY[element][0];
      expect(primarySpell).not.toBeNull();
      expect(primarySpell).not.toBeUndefined();
    }
  });

  it('SPELL_SLOT_REGISTRY covers exactly the same set of keys as ELEMENT', () => {
    const elementKeys = Object.keys(ELEMENT).sort();
    const registryKeys = Object.keys(SPELL_SLOT_REGISTRY).sort();
    expect(registryKeys).toEqual(elementKeys);
  });
});
```
